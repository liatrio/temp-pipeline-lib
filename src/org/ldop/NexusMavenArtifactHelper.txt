<h3>Downloads an artifact from nexus and writes it to targetPath.</h3>
<p>Creates target directories if they don't exist. Errors if the target file already exists</p>
<h4>Parameters:</h4>
<ul>
<li><b>groupId:</b> Artifact group ID
<li><b>artifactId:</b> Artifact ID
<li><b>version:</b> Artifact version
<li><b>targetPath:</b> Local path for the artifact
<li><b>nexus:</b> FQDN + port for nexus
</ul>
